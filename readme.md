# Data Science Machine Learning Proyek Analisis Data

Repository ini berisi proyek analisis data menggunakan teknik data science dan machine learning.

## Cara Menggunakan

1. Clone repository ini:

```sh
git clone https://github.com/yorizpra/data-science_machine-learning_proyek-analisis-data.git
cd data-science_machine-learning_proyek-analisis-data
```

2. Pastikan Anda memiliki Jupyter Notebook atau Jupyter Lab terinstal di sistem Anda.

3. Buka file notebook.ipynb menggunakan Jupyter Notebook atau Jupyter Lab:

```sh
jupyter notebook [notebook.ipynb](http://_vscodecontentref_/1)
```

4. Jalankan setiap sel dalam notebook untuk melihat hasil analisis data.

## Library yang Digunakan
- numpy
- pandas
- matplotlib
- seaborn

## Deskripsi Proyek
Proyek ini melakukan analisis data pada dataset yang terdiri dari informasi pelanggan, pesanan, produk, dan penjualan. Analisis mencakup:

- Pembersihan dan transformasi data
- Eksplorasi data untuk mendapatkan wawasan
- Visualisasi data untuk memudahkan interpretasi

## Sumber Data
Dataset yang digunakan dalam proyek ini diambil dari sumber berikut:

- customers.csv
